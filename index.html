LLM connection

Plan A (default): free browser-based access to an online open-source model via Puter.js — no key, no signup, works out of the box.
Plan B: secure server-side proxy mode. Provider secrets can live on the server, and any optional user-entered key is kept only in memory for this tab instead of being saved in your browser.

Secure mode keeps API keys out of localStorage. If the site owner configures server-side provider keys, visitors never need to enter one. If you choose to enter your own key, it stays only in memory for this tab, is cleared on refresh, and is sent over HTTPS to the server proxy for the current request only.
