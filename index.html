LLM connection

Plan A (default): free browser-based access to an online open-source model via Puter.js — no key, no signup, works out of the box.
Plan B: bring your own API key for a provider you prefer. Keys stay in your browser only.

⚠ Plan A runs through Puter.js in the browser and does not need an API key. For Plan B, your key sits in your browser's localStorage and is sent directly to the provider you chose. Anyone on this device can read it — use a throwaway key with a spend limit for public pages.
